# Gitignore Info

## Files được gitignore (chứa sensitive data):

- `configmap.yaml` - Chứa IP addresses của infrastructure
- `secret.yaml` - Chứa passwords, API keys, tokens

## Files được commit (templates):

- `configmap.yaml.example` - Template cho ConfigMap
- `secret.yaml.example` - Template cho Secret
- `README.md` - Documentation
- `setup.sh` - Setup script

## Lưu ý:

File `configmap.yaml` và `secret.yaml` đã được untrack khỏi Git repository nhưng vẫn tồn tại trên filesystem local. Các file này chứa config thực tế đang chạy trong Kubernetes cluster.

Để sync config từ cluster về local:
```bash
kubectl get configmap -n smap knowledge-config -o yaml > manifests/configmap.yaml
kubectl get secret -n smap knowledge-secrets -o yaml > manifests/secret.yaml
```

Để apply config từ local lên cluster:
```bash
kubectl apply -f manifests/configmap.yaml
kubectl apply -f manifests/secret.yaml
```
